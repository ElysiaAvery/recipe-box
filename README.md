## Setup/Installation Requirements

* In your first terminal window:
  * Start postgres: `$ postgres`
* In your second terminal window:
  * Start psql: `$ psql`
  * Create database: `# CREATE DATABASE recipe_box;`
* In your third terminal window:
  * Clone this repository to your desktop: `$ cd Desktop; git clone https://github.com/CGrahamS/recipe-box`
  * Navigate to the hair-salon directory: `$ cd recipe-box`
  * Create database schema from recipe_box.sql: `$ psql recipe_box < recipe_box.sql`
* Back in your second terminal window:
  * Confirm the database has been restored correctly:
    * Connect to recipe_box database: `# \c recipe_box;`
    * Print out database tables: `# \dt;`
    <br>
    NOTE: You should see `stylists` and `clients` tables in the `recipe_box` database.
* Back in your third terminal window:
  * Run the server: `$ gradle run`
* In the browser of your choosing, navigate to "localhost:4567" (tested in Chrome).
